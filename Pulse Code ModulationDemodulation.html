Pulse Code Modulation/Demodulation

FEATURES

  • In-built sine wave generator
  • Test points to study different waveforms
  • In-built DC Power supply

TECHNICAL SPECIFICATION:

  • Sine wave Generator : Frequency : 1 KHz, Amplitude : 0 to 3Vpp
  • DC Voltage : -1.5 V to +1.5V
  • Clock Generator : Crystal based, 4.096MHz
  • Data rate : 64 kbps (for 8 KHz sampling rate)128kbps (for 16 KHz sampling rate)
  • ADC and DAC bits : 8
  • Demodulator filter type : Butterworth low pass filter
  • Filter type : 6th
  • Filter cut-off frequency : 3.4 KHz
  • Mic Pre-amplifier : Transistor based
  • Companding range : -80dBm to +20dbm
  • Switched faults : 4
  • Test points : 35
  • Fuse : 30V, 100mV
  • DC Supply : ±5V, ±12V
  • System Bandwidth : 3.4 KHz
